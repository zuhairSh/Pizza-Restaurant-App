# 🍕 Pizza Ordering System (WinForms)

A desktop application built with **C#** and **Windows Forms** that allows users to customize and calculate the price of pizza orders through an intuitive graphical interface.

## ✨ Features
*   **Menu Selection:** Easy navigation between the welcome screen and the order menu.
*   **Customization Options:** 
    *   **Size:** Small, Medium, Large.
    *   **Crust Type:** Thin Crust, Thick Crust.
    *   **Toppings:** Extra Cheese, Mushrooms, Onions, Tomatoes.
    *   **Dining Option:** Eat In or Take Out.
*   **Order Summary:** Real-time feedback showing selected options.
*   **Price Calculation:** Automatically calculates the total price based on selections.
*   **Reset Functionality:** Ability to clear the form for a new order.
